# External Accuracy References

These locked fixtures were captured manually from the official Astrodienst
Swiss Ephemeris `swetest` web interface. Unlike `accuracy-charts`, they are not
generated by Astroprocessor and must never be rewritten by
`accuracy:generate`.

Each fixture records:

- the local civil birth data passed to Astroprocessor;
- the resolved UTC instant used in the external command;
- exact `swetest` commands and source metadata;
- tropical geocentric longitudes and longitudinal speeds;
- Koch and Placidus house cusps, Ascendant, and Midheaven.

The audit derives body-to-house placement and major planet aspects independently
from those external numbers.

Run:

```bash
SWISSEPH_EPHE_PATH=/path/to/ephemeris corepack pnpm accuracy:audit
```

Reference files are append-only evidence. To correct or update one:

1. Capture fresh output from the official source.
2. Review timezone and UTC conversion independently.
3. Record the new source version and capture date.
4. Review the fixture diff before committing it.
